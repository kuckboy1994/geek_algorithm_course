# 104. 二叉树的最大深度

1. DFS

* 时间复杂度: O(n)
* 空间复杂度: O(树的高)

```js
/**
 * @param {TreeNode} root
 * @return {number}
 */
var maxDepth = function (root) {
  if (root === null) {
    return 0;
  }

  return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
};
```
